# Краткое описание проекта

## main.go

Назначение: Главная точка входа в приложение
Функции:

+ Инициализация базы данных

+ Запуск веб-сервера

## server.go

Назначение: Конфигурация и запуск HTTP-сервера

Функции:

+ Регистрация API-обработчиков

+ Настройка порта сервера

# Модули API (pkg/api/)
## api.go
Назначение: Регистрация всех API-эндпоинтов

Функции:

+ Связывает URL-пути с соответствующими обработчиками

+ точка настройки маршрутов

## nextdate.go
Назначение: Расчет следующих дат выполнения задач

Функции:

+ Обработка GET /api/nextdate

+ Вычисление следующей даты на основе правил повторения

+ Поддержка различных форматов повторения (ежедневно, ежегодно)

## addtask.go
Назначение: Добавление новых задач

Функции:

+ Обработка POST /api/task

+ Валидация входных данных

+ Преобразование специальных значений дат ("today")

+ Сохранение задач в базу данных

## task.go
Назначение: Работа с отдельными задачами

Функции:

+ Обработка GET /api/task (получение задачи по ID)

+ Обработка PUT /api/task (обновление задачи)

+ Обработка DELETE /api/task (удаление задачи)

## tasks.go
Назначение: Работа со списками задач

Функции:

+ Обработка GET /api/tasks (получение списка задач)

+ Поддержка пагинации через параметр limit

+ Фильтрация и сортировка задач

## task_done.go
Назначение: Отметка выполнения задач

Функции:

+ Обработка POST /api/task/done

+ Для периодических задач - расчет следующей даты

+ Для одноразовых задач - удаление после выполнения

# Модули базы данных (pkg/db/)
## db.go
Назначение: Инициализация и управление базой данных

Функции:

+ Подключение к SQLite базе данных

+ Создание таблиц и индексов при первом запуске

+ Управление соединением с БД

## task.go
Назначение: Работа с данными задач в БД

Функции:

+ Определение структуры Task

+ CRUD-операции для задач (создание, чтение, обновление, удаление)

+ Специализированные запросы к базе данных
